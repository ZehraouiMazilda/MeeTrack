# ● LUMI — Assistant d'étude intelligent

Lumi est une application web d'assistance à l'étude combinant :
- 📷 Détection de concentration par caméra (MediaPipe)
- 🎙 Assistant vocal avec wake word "Lumi" (Whisper + Groq)
- 💬 Chatbot contextuel basé sur tes cours (Llama 3.1)
- 📊 Analytics de session

**Stack :** Streamlit · Groq · MediaPipe · OpenCV · SQLite
